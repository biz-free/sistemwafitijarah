🔧 Panduan Setup Supabase
Ikut langkah ini untuk sambung database cloud percuma
1
Daftar Supabase — Pergi ke supabase.com → klik "Start your project" → log masuk dengan Google
2
Buat Project Baru — Nama: wafi-tijarah, Region: Southeast Asia (Singapore), tetapkan kata laluan database
3
Salin URL & Key — Pergi ke Settings → API, salin Project URL dan anon/public key
4
Jalankan SQL penuh dalam SQL Editor Supabase — salin dari fail PANDUAN_SETUP.md (jadual, RLS policy & fungsi RPC untuk stok/hutang yang selamat & atomik)
5
Cipta akaun pekerja — Pergi Authentication → Users → "Add user" untuk setiap pekerja/pemilik, salin User UID masing-masing, kemudian masukkan baris ke jadual profiles (id = UID, role = 'pemilik'/'pekerja', nama)
6
Masukkan credentials dalam aplikasi — Buka fail index.html, cari SUPABASE_URL & SUPABASE_KEY di bahagian atas <script>, gantikan dengan URL & anon key projek anda, kemudian upload semula
✅ Supabase percuma sehingga 50,000 baris data & 2GB storage — lebih dari cukup untuk perniagaan Wafi Tijarah!
⚠️ Selepas sambung Supabase, akaun awal (pemilik@wafi.com dll.) tak lagi berfungsi — log masuk guna akaun yang anda cipta di Langkah 5.
